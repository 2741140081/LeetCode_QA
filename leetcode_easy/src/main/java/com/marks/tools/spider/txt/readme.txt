这个文件夹下的java 类主要用于爬取小说内容并且将其保存到本地
1. 保存的目录 D:/spider/txt/<小说名称>
2. 每一个页面对于小说中的一个章节, 可以通过目录进行拼接获取网页地址信息
3. 将获取到的地址进行https 访问, 通过代码得到 Doc 类, 包含了该网页中的信息
4. 由于章节页面的结构是类似的, 即小说章节的内容全部在网页的特定结构下放置(待配置)
5. 小说章节的内容主体部分由于排版需要, 会添加 <br> 进行换行操作, 这些特殊的标识符不需要保存到txt 文件中
6. 只需要提取<p> 中的内容, 这个提取出来的内容是否是有序(待验证)
7. 使用多线程进行优化, 多线程处理两个部分, 解析页面部分 和 将小说的章节内容保存为txt文件, 分别配置两个线程池进行处理
