差分数组是一种用于高效处理数组区间更新的数据结构，核心价值是将区间批量修改的时间复杂度从O(n)优化至O(1)‌，以下是具体定义和用途说明：

一、差分数组的定义

对于长度为n的原数组A，对应的差分数组D满足：

若下标从0开始：D = A，对于i > 0，D[i] = A[i] - A[i-1]；
若下标从1开始：D = A，对于i > 1，D[i] = A[i] - A[i-1]。

核心性质：原数组可以通过差分数组的‌前缀和还原‌，即原数组第i项A[i] = D + D + ... + D[i]，差分数组和原数组互为逆操作：差分数组是原数组的差值，原数组是差分数组的前缀和。

举个简单例子：原数组A = [1, 3, 6, 10, 15]，计算得到的差分数组为D = [1, 2, 3, 4, 5]，通过D的前缀和可以完美还原出A。

二、差分数组的核心用途

差分数组的核心优势是‌空间换时间‌，适合处理「先批量更新，后统一查询」的离线场景，主要用途分为两类：

快速处理区间加减操作‌
当需要对原数组的区间[L, R]内所有元素统一加上数值x时，无需遍历区间内每个元素，仅需修改差分数组的两个位置：

D[L] += x（从L开始，所有元素的前缀和都会自动增加x）
若R+1 < n，则D[R+1] -= x（抵消R之后元素的额外增量，保证不影响区间外元素）
整个操作仅需O(1)时间复杂度，当存在大量区间更新操作时，效率提升非常显著。

配合前缀和解决区间和查询问题‌
在完成所有区间更新后，先通过差分数组的前缀和还原出最终的原数组，再计算原数组的前缀和数组，即可快速得到任意区间[L, R]的区间和：区间和 = 前缀和[R] - 前缀和[L-1]，能够快速响应区间和查询需求。

三、常见应用场景

差分数组在实际开发和算法题中应用广泛，典型场景包括：

日志统计：统计某时间段内的用户活跃数变化
资源分配：动态调整区间内的资源配额
游戏开发：批量修改区域内的角色属性
图像处理：二维差分数组批量修改图像像素区域
算法题：区间染色、航班预订统计、批量区间修改等经典问题
